# arabic-answer Overlay — WAVE 2B Invocation Instructions

## Server Name (from mcp_servers.json)
`super-agents`

## How the arabic-answer Overlay Works

The overlay file lives at:
  src/templates/overlays/arabic-answer.mdx
  build/templates/overlays/arabic-answer.mdx

The template loader uses a two-step fallback:
1. First tries: overlays/{role-prefix}-arabic-answer.mdx  (e.g. coder-arabic-answer.mdx)
2. Falls back to: overlays/arabic-answer.mdx  ← THIS IS THE FILE THAT WILL LOAD

This means `arabic-answer` works as a cross-role specialization for ANY role.

## Exact Invocation (MCP tool call)

```json
{
  "tool": "spawn_agent",
  "arguments": {
    "role": "coder",
    "prompt": "<your task brief here>",
    "specialization": "arabic-answer"
  }
}
```

Works for ALL roles (coder, planner, tester, researcher):
- role: "coder"      + specialization: "arabic-answer"
- role: "planner"    + specialization: "arabic-answer"
- role: "tester"     + specialization: "arabic-answer"
- role: "researcher" + specialization: "arabic-answer"

## Effect
- All file output written in Arabic script
- Numbers in prose use Arabic words (واحد through عشرة)
- Every output file closes with: تم بواسطة الوكيل
- Role methodology (tools, reasoning, workflow) unchanged
