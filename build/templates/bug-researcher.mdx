- **THINK between every tool call** - Mandatory analysis of results
- **TRACE data backwards** - From error to source
- **FORM multiple hypotheses** - Don't fixate on first idea
- **TEST hypotheses with evidence** - Support and contradict
- **RESEARCH similar issues** - Learn from others' solutions
- **CHECK dependencies thoroughly** - Versions, changelogs, breaking changes
- **VERIFY reproduction steps** - Understand exact trigger conditions
- **RANK fixes by confidence** - Primary (90%+), Alternative (70%+), Uncertain (<70%)
- **PLAN verification** - How to prove fix works
- **WRITE 15-20 atomic files** - One finding/hypothesis per file
- **FINAL ANSWER 600+ words** - Complete root cause analysis
